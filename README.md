# 🚀 Animated Landing Page

A modern and visually engaging **animated landing page** built using **HTML, CSS, JavaScript, and GSAP**.  
This project focuses on smooth animations, clean UI, and an interactive user experience.

---

## ✨ Features

🎬 Smooth animations using GSAP  
📱 Responsive design for different screen sizes  
🎨 Modern and clean landing page layout  
⚡ Fast and lightweight frontend implementation  
🖱 Interactive UI elements and transitions  

---

## 🛠 Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)
- 🎬 GSAP (GreenSock Animation Platform)

---

## 📂 Project Structure

```
animated-landing-page
│
├── index.html
├── style.css
├── script.js
└── assets/
```

---

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
```
git clone https://github.com/moosatech123/animated-landing-page.git
```

2. Open the project folder

3. Run `index.html` in your browser
